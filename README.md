# Intelligent Routing Protocols for Smart Traffic and Urban Mobility Networks

> "Conventional vehicular routing is broken. Build for the architecture
> that exists in 5 years, not the one that existed 5 years ago."

## What This Paper Is About

Cities are choking. Vehicular density is surging. Legacy traffic
management is failing in real time. The fix is not wider roads —
it is smarter networks.

This paper is a first-principles analysis of routing protocols for
vehicular networks. No padding. No hedging. Just what works, what
doesn't, and what's next.

## Four Protocol Families. One Winner Per Context.

| Protocol | Verdict |
|----------|---------|
| Topology-Based (AODV, DSDV) | Legacy. Collapses above 50 vehicles/km |
| Geographic (GPSR, GyTAR) | Best current option. Road-aware is mandatory |
| Broadcast / Geocast | Safety-critical only. Storm suppression required |
| Learning-Based (RL, DNN) | The future. Sim-to-real gap is the bottleneck |

## Real-World Impact

- **20–35% reduction** in travel time on green wave corridors
- **Sub-100ms** hazard warning delivery for safety-critical apps
- **City-scale** freight optimisation through real-time routing
